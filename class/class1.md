### 0.3 C언어와 프로그램 실행 흐름 (처음 보는 학생용)

- **C언어**는 사람이 작성한 **소스 코드**를 컴퓨터가 이해하는 **실행 파일**로 바꿔 주는 **컴파일러**를 통해 실행됩니다.
- **소스 파일**: `.c`(C 전용) 또는 `.cpp`(C++ 확장, C 문법도 사용 가능)로 저장합니다. ex240311은 **ex1.cpp**에 C 문법으로 작성된 예제가 들어 있습니다.
- **실행 순서**:  
  1. 소스 코드 작성 → 2. **컴파일**(빌드) → 3. **실행**  
  IDE에서는 보통 [실행] 또는 [디버깅 시작] 버튼으로 2번과 3번이 한 번에 진행됩니다.
- 프로그램이 시작되면 **항상 `main()` 함수**가 먼저 호출되고, 그 안의 문장이 **위에서 아래로 순서대로** 실행됩니다.

### 0.4 주석이란? (코드에 설명 넣기)

Main.cpp에는 `//연산자 : 산술 연산자`, `//헤더파일 정의`, `//main() 함수 정의`, `//명령 정의`, `// X` 같은 **주석**이 있습니다. 주석은 **컴파일되지 않고**, 사람이 읽기 위한 설명입니다.

| 방식 | 예시 | 설명 |
|------|------|------|
| **한 줄 주석** | `// 산술 연산자 + - * / %` | `//` 뒤부터 그 줄 끝까지가 주석입니다. |
| **여러 줄 주석** | `/* 이 부분은 실행되지 않습니다 */` | `/*`와 `*/` 사이 전체가 주석입니다. |

**예시 (Main.cpp 스타일)**
```c
#include <stdio.h>  // 헤더파일 정의

int main()          // main() 함수 정의
{
  printf("Hello!\n\n\n");  // 명령 정의
}
```

- 연습할 때 **일부 코드를 주석 처리**(앞에 `//` 붙이기)하면 그 부분만 실행에서 제외되므로, 예제를 하나씩 비교해 보기에 좋습니다.

## 1. C 프로그램의 기본 구조

### 1.1 가장 단순한 프로그램

```c
#include <stdio.h>

int main()
{
  printf("Hello");
}
```

**설명**
- `#include <stdio.h>` : 화면에 출력하는 등 **표준 입출력**을 쓰기 위해 필요한 헤더 파일을 불러옵니다. 거의 모든 C 프로그램에서 사용합니다.
- `int main()` : 프로그램이 **시작되는 곳**입니다. 실행 시 이 함수부터 실행됩니다.
- `{ }` : 중괄호 안에 실행할 **명령(문장)**들을 씁니다.
- `printf("Hello");` : 따옴표 안의 문자열 **"Hello"**를 화면에 출력합니다. 문장 끝에는 반드시 `;`(세미콜론)을 붙입니다.

---

### 1.2 줄바꿈과 여러 번 출력

```c
#include <stdio.h>

int main()
{
  printf("Hello!\n\n\n");
  printf("Hello!\n\n\n");
  printf("Hello!\n\n\n");
}
```

**설명**
- `\n` : **줄바꿈**을 의미하는 특수 문자(이스케이프 시퀀스)입니다.
- `\n\n\n`이면 줄바꿈이 세 번 일어나서, 출력 사이에 빈 줄이 생깁니다.

**예상 출력**
```
Hello!



Hello!



Hello!


```

---

### 1.3 번호를 붙여서 출력하기

```c
#include <stdio.h>

int main()
{
  printf("1. Hello!\n");
  printf("2. Hello!\n");
  printf("3. Hello!\n");
}
```

**설명**
- 문자열 안에 숫자나 문장을 넣어서 **순서대로** 출력할 수 있습니다.

---

### 1.4 getchar()로 실행 순서 이해하기

```c
#include <stdio.h>

int main()
{
  printf("1. Hello!\n");
  printf("2. Hello!\n");
  printf("3. Hello!\n");
  getchar();           // 여기서 키보드 입력 한 글자를 기다림
  printf("4. Hello!\n");
  printf("5. Hello!\n");
}
```

**설명**
- `getchar()` : **키보드에서 문자 하나를 입력받을 때까지** 프로그램이 멈춥니다.
- 1, 2, 3이 출력된 뒤, 사용자가 아무 키나 누르면 4, 5가 출력됩니다.
- C에서는 **위에서 아래로 순서대로** 문장이 실행된다는 것을 확인할 수 있는 예제입니다.

## 2. printf와 형식 지정자(서식 문자)

`printf`에서 숫자나 문자를 **형식에 맞게** 넣어 출력하려면 **형식 지정자**를 사용합니다.

### 2.1 정수 출력: %d

```c
#include <stdio.h>

int main()
{
  printf("Hello : %d\n", 10);
  printf("Hello : %d %d\n", 10, 20);
  printf("Hello : %d %d %d\n", 10, 20, 30);
}
```

**설명**
- `%d` : **정수(decimal)**를 10진수로 출력합니다.
- 문자열 안의 `%d` 개수만큼, 뒤에 **쉼표로 구분된 값**을 순서대로 넣습니다.
  - 첫 번째 `%d` ← 10, 두 번째 `%d` ← 20, 세 번째 `%d` ← 30

**예상 출력**
```
Hello : 10
Hello : 10 20
Hello : 10 20 30
```

---

### 2.2 문자 출력: %c

```c
#include <stdio.h>

int main()
{
  printf("Hello : %c\n", 'A');
  printf("Hello : %d\n", 'A');
  printf("Hello : %c\n", 65);
  printf("Hello : %d\n", 65);
}
```

**설명**
- `%c` : **한 글자(문자)**를 출력합니다. 문자는 **작은따옴표** `'A'`로 씁니다.
- C에서는 문자도 내부적으로 **숫자(코드)**로 저장됩니다. 'A'는 ASCII 코드로 **65**입니다.
- 따라서 `%c`에 65를 주면 'A'가 출력되고, `%d`에 'A'를 주면 65가 출력됩니다.

**예상 출력**
```
Hello : A
Hello : 65
Hello : A
Hello : 65
```

---

### 2.3 실수 출력: %f, %lf, %e, %g

```c
#include <stdio.h>

int main()
{
  printf("Hello : %f\n", 1.2);
  printf("Hello : %lf\n", 1.2);
  printf("Hello : %e\n", 1.2);
  printf("Hello : %g\n", 1.2);
}
```

**설명**
- `%f` : 실수를 **일반 소수 형태**로 출력 (float, double 모두 사용 가능).
- `%lf` : **double**용 실수 출력에 자주 씁니다.
- `%e` : **과학적 표기법** (예: 1.2e+00).
- `%g` : 값에 따라 **짧게** 보기 좋은 형태로 출력합니다.

---

### 2.4 문자열 출력: %s

```c
#include <stdio.h>

int main()
{
  printf("Hello : %s\n", "1.2");
  printf("Hello : %s\n", "Hello");
}
```

**설명**
- `%s` : **문자열**을 출력합니다. 문자열은 **큰따옴표** `"Hello"`로 씁니다.
- `"1.2"`는 숫자가 아니라 **글자 네 개**('1', '.', '2', 끝 표시)로 취급됩니다.

---

### 2.5 변수에 저장한 값 출력

```c
#include <stdio.h>

int main()
{
  int n = 10;
  printf("Hello : %d\n", n);
  printf("Hello : %d\n", 10);
}
```

**설명**
- `int n = 10;` : **정수형 변수** `n`을 만들고 10을 저장합니다.
- `printf`에서 `%d` 자리에 **변수 이름** `n`을 넣으면, 그 변수에 들어 있는 값이 출력됩니다.
- `n`과 `10`을 각각 출력하면 여기서는 같은 10이 나옵니다.

---

### 2.6 double 변수와 %g

```c
#include <stdio.h>

int main()
{
  double d = 10.12;
  printf("Hello : %g\n", d);
  printf("Hello : %g\n", 10.12);
}
```

**설명**
- `double d = 10.12;` : **실수형 변수** `d`를 만들고 10.12를 저장합니다.
- `%g`로 실수를 출력하면 보통 **불필요한 0**을 줄여서 보여줍니다.

---

### 2.7 char 변수와 문자 출력

```c
#include <stdio.h>

int main()
{
  char c = 'A';
  printf("Hello : %c\n", c);
  printf("Hello : %c\n", 'A');
}
```

**설명**
- `char c = 'A';` : **문자형 변수** `c`를 만들고 문자 'A'를 저장합니다.
- `%c`로 출력하면 **문자**로 보이고, `%d`로 출력하면 그 문자의 **코드 값(숫자)**이 나옵니다.

## 3. 산술 연산자 (+, -, *, /, %)

ex240311의 **메인 예제**는 **정수의 사칙연산과 나머지**입니다.

### 3.1 변수를 사용한 연산 (교재 메인 코드)

```c
#include <stdio.h>

int main()
{
  // 산술 연산자 : 더하기 + 빼기 - 곱하기 * 나누기 / 나머지 %

  int a = 5;
  int b = 3;
  int result = 0;

  result = a + b;
  printf(" %d \n", result);
  result = a - b;
  printf(" %d \n", result);
  result = a * b;
  printf(" %d \n", result);
  result = a / b;
  printf(" %d \n", result);
  result = a % b;
  printf(" %d \n", result);
}
```

**설명**
- `int a = 5;`, `int b = 3;` : 정수 변수 `a`, `b`에 5와 3을 저장합니다.
- `int result = 0;` : 연산 결과를 담을 변수 `result`를 0으로 초기화합니다.
- **연산자**
  - `+` : 더하기 → 5 + 3 = **8**
  - `-` : 빼기 → 5 - 3 = **2**
  - `*` : 곱하기 → 5 * 3 = **15**
  - `/` : 나누기(몫) → 5 / 3 = **1** (정수끼리 나누면 소수 이하는 버려짐)
  - `%` : 나머지 → 5 % 3 = **2**

**예상 출력**
```
 8
 2
 15
 1
 2
```

---

### 3.2 변수 없이 식을 직접 출력

```c
#include <stdio.h>

int main()
{
  int a = 5;
  int b = 3;

  printf(" %d \n", a + b);
  printf(" %d \n", a - b);
  printf(" %d \n", a * b);
  printf(" %d \n", a / b);
  printf(" %d \n", a % b);
}
```

**설명**
- `result` 변수 없이, `printf` 안에서 바로 `a + b`, `a - b` 등의 **식**을 계산해 출력할 수 있습니다.
- 결과는 위와 동일합니다.

---

### 3.3 정수 나눗셈 주의점

- C에서 **정수 / 정수**는 결과도 **정수**입니다. 소수점 이하는 **버려집니다**.
  - 예: `5 / 3` → 1 (1.666…이 아니라 1만 저장)
- **나머지**는 `%` 연산자로 구합니다. `5 % 3` → 2.

**예제**
```c
printf("%d\n", 10 / 3);   // 3
printf("%d\n", 10 % 3);   // 1
```

## 4. 자료형과 크기 (sizeof)

### 4.1 sizeof로 크기 확인

```c
#include <stdio.h>

int main()
{
  char c = 10;
  int n = 10;

  printf("%d %d \n", sizeof(char), sizeof(int));
  printf("%d %d \n", sizeof(c), sizeof(n));
}
```

**설명**
- `sizeof(타입)` 또는 `sizeof(변수)` : 해당 타입/변수가 메모리에서 **몇 바이트**를 쓰는지 알려줍니다.
- **char** : 보통 **1바이트** (한 글자 저장).
- **int** : 보통 **4바이트** (정수 저장).
- 환경에 따라 int는 2바이트일 수도, 4바이트일 수도 있지만, 요즘 대부분의 환경에서는 4입니다.

**예상 출력 (일반적인 32/64비트 환경)**
```
1 4
1 4
```

## 5. char와 int의 차이 (범위와 오버플로우)

### 5.1 char에 큰 수를 넣으면?

```c
#include <stdio.h>

int main()
{
  char c = 10;
  int n = 10;

  printf("%d %d \n", c, n);

  c = 20;
  n = 20;
  printf("%d %d \n", c, n);

  c = 500;  // X - char 범위를 벗어남 (오버플로우)
  n = 500;
  printf("%d %d \n", c, n);
}
```

**설명**
- **char**는 1바이트라서 보통 **-128 ~ 127** (또는 0 ~ 255) 범위만 안전하게 저장할 수 있습니다.
- **500**은 이 범위를 넘어갑니다. 그래서 `c = 500`은 **오버플로우**가 나고, 실제로는 이상한(의도하지 않은) 값이 저장됩니다.
- **int**는 4바이트라서 500을 정상적으로 저장하고, 500이 그대로 출력됩니다.

**정리**
- 작은 정수만 쓸 때는 `char`도 쓸 수 있지만, **범위**를 반드시 생각해야 합니다.
- 일반적인 정수는 `int`를 쓰는 것이 안전합니다.

---

## 5.2 char의 signed / unsigned (보충)

- **char**는 환경에 따라 **signed char**(부호 있음, 보통 -128 ~ 127) 또는 **unsigned char**(부호 없음, 0 ~ 255)로 취급될 수 있습니다.
- **500**을 넣으면 두 경우 모두 **범위를 벗어나** 오버플로우가 발생합니다. 정수 계산용으로는 **int**를 쓰는 것이 안전합니다.

### 6.1 이스케이프 시퀀스 (특수 문자)

문자열 안에서 줄바꿈·탭 등 **특수한 문자**를 나타낼 때 `\`로 시작하는 **이스케이프 시퀀스**를 씁니다.

| 표기 | 의미 |
|------|------|
| `\n` | 줄바꿈 (개행) |
| `\t` | 탭 (들여쓰기) |
| `\\` | 역슬래시 하나 `\` |
| `\"` | 큰따옴표 하나 `"` (문자열 안에서 사용) |
| `\'` | 작은따옴표 하나 `'` |

**예시**
```c
printf("이름\t점수\n");
printf("홍길동\t85\n");
```

---

### 6.2 변수 선언과 초기화

- **선언만 하는 경우**: `int result;` → 쓰레기 값이 들어 있을 수 있어, 사용 전에 반드시 값을 넣어 주는 것이 좋습니다.
- **선언과 동시에 초기화**: `int result = 0;` → Main.cpp의 `int result = 0;`처럼 **초기화**하면 예측 가능하게 사용할 수 있습니다.
- **여러 변수 한 줄에**: `int a = 5, b = 3;`처럼 쉼표로 구분해 쓸 수 있습니다. (가독성을 위해 한 줄에 하나씩 쓰는 것도 좋습니다.)

---

### 6.3 형식 지정자와 인자 개수

- `printf("서식 문자열", 값1, 값2, ...)`에서 **서식 문자열 안의 형식 지정자 개수**와 **뒤에 오는 값의 개수**가 **같아야** 합니다.
- 개수가 맞지 않으면 컴파일 경고가 나거나, 실행 시 잘못된 값이 출력될 수 있습니다.

**올바른 예**
```c
printf("Hello : %d %d\n", 10, 20);   // %d 2개, 값 2개
```

**주의할 예**
```c
printf("Hello : %d %d\n", 10);       // %d 2개인데 값 1개 → 오류/경고
```

---

### 6.4 float와 double의 차이 (실수형 보충)

Main.cpp에는 `%f`, `%lf`, `%e`, `%g`와 `double d = 10.12;` 예제가 있습니다.

| 타입 | 크기(대략) | 정밀도 | 형식 지정자 |
|------|------------|--------|-------------|
| **float** | 4바이트 | 소수점 이하 약 6자리 정도 | `%f` |
| **double** | 8바이트 | 소수점 이하 약 15자리 정도 | `%lf` 또는 `%f` |

- **double**이 **float**보다 범위와 정밀도가 크므로, 실수를 쓸 때는 **double**을 쓰는 경우가 많습니다.
- `printf`에서 `%f`는 float와 double 모두 받을 수 있지만, **scanf**에서는 `float`는 `%f`, `double`는 `%lf`를 사용해야 합니다.

---

### 6.5 문자와 숫자 (ASCII 코드) 정리

Main.cpp에서는 `'A'`와 `65`를 `%c`, `%d`로 출력하는 예제가 있습니다. 문자는 내부적으로 **정수(코드 값)**로 저장됩니다.

| 문자 | ASCII 코드(10진) | 비고 |
|------|------------------|------|
| `'A'` | 65 | 대문자 A |
| `'a'` | 97 | 소문자 a |
| `'0'` | 48 | 숫자 문자 0 (정수 0과 다름) |
| `' '` | 32 | 공백 |

- **숫자 0**과 **문자 '0'**은 다릅니다. `printf("%d", 0);` → 0, `printf("%d", '0');` → 48.
- `'A' + 1`은 정수 66이므로, `%c`로 출력하면 `'B'`가 됩니다.

---

### 6.6 .c 파일과 .cpp 파일

- **.c** : C 전용 소스 파일. C 컴파일러로 컴파일합니다.
- **.cpp** (또는 .cc 등) : C++ 소스 파일 확장자입니다. C++ 컴파일러는 **C 문법도 대부분 지원**하므로, ex240311처럼 **C만 사용한 코드**를 .cpp에 넣어도 동작합니다.
- C만 배우는 단계에서는 **문법을 C로 통일**하고, 파일 확장자는 사용하는 환경(.c 또는 .cpp)에 맞추면 됩니다.
