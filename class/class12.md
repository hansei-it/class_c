## 1. 조건문 (if/else, switch)

조건문은 **조건에 따라 실행할 코드를 선택**하는 구문입니다. 프로그램이 항상 위에서 아래로만 흐르지 않고, 상황에 따라 **다른 길**을 갈 수 있게 합니다.

### 1.1 if / else if / else – 성적 판별

```c
#include <stdio.h>
int main()
{
	int score = 10;

	if (score >= 90)
		printf("A\n");
	else if (score >= 80)
		printf("B\n");
	else if (score >= 70)
		printf("C\n");
	else
		printf("F\n");
}
```

**설명**
- `if (조건)` : 조건이 **참(0이 아닌 값)**이면 바로 아래 문장을 실행합니다.
- `else if (조건)` : 위의 `if`가 거짓일 때, **추가 조건**을 확인합니다.
- `else` : 위의 모든 조건이 거짓일 때 실행됩니다.
- 이 예제에서 `score = 10`이므로, 90 이상도, 80 이상도, 70 이상도 아니어서 `else`에 도달하여 **"F"**가 출력됩니다.
- 조건은 **위에서부터 순서대로** 검사하고, 참인 조건을 만나면 해당 블록만 실행한 뒤 나머지는 건너뜁니다.

**예상 출력**
```
F
```

---

### 1.2 switch / case – 성적 판별

```c
#include <stdio.h>
int main()
{
	int score = 95;

	switch (score / 10)
	{
	case 9:
		printf("A\n");
		break;
	case 8:
		printf("B\n");
		break;
	case 7:
		printf("C\n");
		break;
	default:
		printf("F\n");
		break;
	}
}
```

**설명**
- `switch (식)` : 식의 **결과 값**과 일치하는 `case`로 이동하여 실행합니다.
- `score / 10` : 정수 나눗셈이므로 95 / 10 = **9**가 됩니다.
- `case 9:` → `"A"` 출력 후 `break`로 switch를 빠져나옵니다.
- `break`가 없으면 아래 case로 **연달아 실행**(fall-through)되므로 주의해야 합니다.
- `default:` : 어떤 case에도 해당하지 않을 때 실행됩니다 (`else`와 비슷).
- `if/else if`와 같은 결과를 낼 수 있지만, **정수 값으로 분기**할 때 `switch`가 깔끔합니다.

**예상 출력**
```
A
```

## 2. 반복문 (for, while, do-while)

같은 작업을 여러 번 수행해야 할 때 **반복문**을 사용합니다.

### 2.1 반복 출력의 필요성 – 같은 코드를 5번 쓰기

```c
#include <stdio.h>
int main()
{
	printf("int : %d\n", 1);
	printf("int : %d\n", 2);
	printf("int : %d\n", 3);
	printf("int : %d\n", 4);
	printf("int : %d\n", 5);
}
```

**설명**
- 1부터 5까지 출력하기 위해 `printf`를 **5번** 직접 씁니다.
- 만약 100번이나 1000번 출력해야 한다면? 이 방식은 비효율적입니다.
- 이런 **반복 작업**을 자동화하기 위해 **반복문**을 사용합니다.

**예상 출력**
```
int : 1
int : 2
int : 3
int : 4
int : 5
```

---

### 2.2 for 루프 기본 (1부터 시작)

```c
#include <stdio.h>
int main()
{
	for(int i = 1 ; i <= 5 ; ++i)
		printf("int : %d\n", i);
}
```

**설명**
- `for (초기식; 조건식; 증감식)` : 반복문의 세 요소를 한 줄에 씁니다.
  - **초기식** `int i = 1` : 반복 변수 `i`를 1로 시작합니다.
  - **조건식** `i <= 5` : `i`가 5 이하인 동안 반복합니다.
  - **증감식** `++i` : 한 번 반복할 때마다 `i`를 1 증가시킵니다.
- 위의 5줄짜리 코드와 **동일한 결과**를 2줄로 작성할 수 있습니다.
- `++i`는 `i = i + 1`과 같은 의미입니다.

**예상 출력**
```
int : 1
int : 2
int : 3
int : 4
int : 5
```

---

### 2.3 for 루프 기본 (0부터 시작)

```c
#include <stdio.h>
int main()
{
	for (int i = 0; i < 5; ++i)
		printf("int : %d\n", i+1);
}
```

**설명**
- 프로그래밍에서는 **0부터 시작**하는 것이 관례입니다 (배열 인덱스 등).
- `i = 0`부터 시작하고 `i < 5`로 조건을 주면 `i`는 0, 1, 2, 3, 4 총 5번 반복합니다.
- 출력 시 `i+1`을 하면 1, 2, 3, 4, 5가 출력됩니다.
- 결과는 2.2와 동일하지만, **0 기반 인덱스**에 익숙해지는 것이 중요합니다.

**예상 출력**
```
int : 1
int : 2
int : 3
int : 4
int : 5
```

---

### 2.4 while 루프

```c
#include <stdio.h>
int main()
{
	int n = 10;
	while (n < 5)
	{
		printf("int : %d\n", n);
		++n;
	}
}
```

**설명**
- `while (조건)` : 조건이 **참인 동안** 중괄호 안의 코드를 반복 실행합니다.
- 이 예제에서 `n = 10`이고 조건은 `n < 5`입니다. **처음부터 조건이 거짓**이므로, 중괄호 안의 코드는 **한 번도 실행되지 않습니다**.
- `while`은 조건을 **먼저 검사**한 뒤 실행 여부를 결정합니다.

**예상 출력**
```
(아무것도 출력되지 않음)
```

---

### 2.5 do-while 루프

```c
#include <stdio.h>
int main()
{
	int n = 10;
	do
	{
		printf("int : %d\n", n);
		++n;
	} while (n < 5);
}
```

**설명**
- `do { ... } while (조건);` : 코드를 **먼저 한 번 실행**한 뒤, 조건을 검사합니다.
- `n = 10`이고 조건 `n < 5`는 거짓이지만, **최소 한 번은 실행**됩니다.
- `while`과의 차이: `while`은 조건이 거짓이면 한 번도 실행 안 되지만, `do-while`은 **반드시 한 번은 실행**됩니다.
- `do-while` 끝에 **세미콜론(`;`)**을 반드시 붙여야 합니다.

**예상 출력**
```
int : 10
```

---

### 2.6 while과 do-while 비교 정리

| 구분 | while | do-while |
|------|-------|----------|
| **조건 검사 시점** | 반복 **전** | 반복 **후** |
| **최소 실행 횟수** | 0번 (조건이 처음부터 거짓이면 미실행) | 1번 (무조건 한 번 실행) |
| **사용 예** | 반복 횟수가 0일 수도 있는 경우 | 최소 한 번은 실행해야 하는 경우 (메뉴 표시 등) |

## 3. break와 continue

반복문 안에서 **흐름을 제어**하는 두 가지 키워드입니다.

### 3.1 for 루프에서 조건 필터링 – 짝수만 출력

```c
#include <stdio.h>
int main()
{
	int size = 10;

	for (int i = 0; i < size; ++i)
	{
		if (i % 2 == 0)
			printf("%d\n", i);
	}
}
```

**설명**
- `i % 2 == 0` : `i`를 2로 나눈 나머지가 0이면 **짝수**입니다.
- 0부터 9까지 반복하면서 짝수인 경우에만 `printf`를 실행합니다.
- 이 방식은 `break`나 `continue` 없이 **`if`만으로 필터링**하는 방법입니다.

**예상 출력**
```
0
2
4
6
8
```

---

### 3.2 continue로 건너뛰기 – 3의 배수만 출력

```c
#include <stdio.h>
int main()
{
	int size = 10;

	for (int i = 0; i < size; ++i)
	{
		if (i % 3)
			continue;

		printf("%d\n", i);
	}
}
```

**설명**
- `continue` : 현재 반복의 **남은 코드를 건너뛰고**, 다음 반복으로 이동합니다.
- `i % 3`의 결과가 **0이 아니면(참이면)** `continue`가 실행되어 `printf`를 건너뜁니다.
- `i % 3`이 **0이면(거짓이면)** `continue`를 지나쳐서 `printf`가 실행됩니다.
- 즉, **3의 배수**(0, 3, 6, 9)만 출력됩니다.
- C에서 **0은 거짓**, **0이 아닌 값은 참**으로 취급됩니다.

**예상 출력**
```
0
3
6
9
```

---

### 3.3 무한 루프와 break – 사용자 입력 받기

```c
#include <stdio.h>
int main()
{
	while (1)
	{
		int data = 0;
		printf("정수 입력: ");
		scanf_s("%d", &data);

		if (data >= 0)
			printf("int : %d\n", data);
		else
			break;
	}
}
```

**설명**
- `while (1)` : 조건이 항상 **참(1)**이므로 **무한 루프**입니다.
- `scanf_s("%d", &data)` : 키보드에서 **정수를 입력**받아 `data`에 저장합니다. (`scanf_s`는 Visual Studio에서 사용하는 안전한 버전입니다. 다른 환경에서는 `scanf`를 사용합니다.)
- 입력값이 **0 이상**이면 출력하고, **음수**면 `break`로 루프를 **즉시 탈출**합니다.
- `break`는 자신을 감싸는 **가장 가까운 반복문**을 빠져나갑니다.

**예상 출력 (입력: 5, 10, -1)**
```
정수 입력: 5
int : 5
정수 입력: 10
int : 10
정수 입력: -1
```

---

### 3.4 while + flag 변수로 반복 제어

```c
#include <stdio.h>
int main()
{
	int flag = 1;

	while (flag)
	{
		int data = 0;
		printf("정수 입력: ");
		scanf_s("%d", &data);

		if (data >= 0)
			printf("int : %d\n", data);
		else
			flag = 0;
	}
}
```

**설명**
- `flag`라는 변수를 사용하여 반복 여부를 제어합니다.
- `flag = 1`(참)인 동안 반복하고, 음수가 입력되면 `flag = 0`(거짓)으로 바꿔서 루프를 종료합니다.
- 3.3의 `while(1) + break`와 **동일한 동작**을 합니다.
- 두 방식 모두 자주 쓰이며, 상황에 따라 편한 쪽을 선택합니다.

---

### 3.5 break와 continue 비교

| 키워드 | 동작 |
|--------|------|
| `break` | 반복문을 **즉시 종료**하고, 반복문 다음 코드로 이동 |
| `continue` | 현재 반복의 나머지를 **건너뛰고**, 다음 반복으로 이동 |

## 4. 함수 (선언, 매개변수, 반환값)

**함수**는 특정 작업을 하나로 묶어 **이름을 붙인 코드 블록**입니다. 반복되는 코드를 줄이고, 프로그램을 구조적으로 만들 수 있습니다.

### 4.1 반복되는 코드 – 함수가 필요한 이유

```c
#include <stdio.h>
int main()
{
	printf("Hello!\n");
	printf("Hello!\n");
	printf("Hello!\n");
}
```

**설명**
- `"Hello!\n"`를 3번 출력합니다.
- 만약 이 3줄짜리 출력을 프로그램 여러 곳에서 사용해야 한다면, 매번 3줄을 반복해서 써야 합니다.
- 이런 반복을 **함수**로 묶으면 코드를 재사용할 수 있습니다.

---

### 4.2 매개변수 없는 함수 – PrintHello()

```c
#include <stdio.h>

void PrintHello()
{
	printf("Hello!\n");
	printf("Hello!\n");
	printf("Hello!\n");
}
int main()
{
	PrintHello();
	PrintHello();
}
```

**설명**
- `void PrintHello()` : **반환값이 없는(`void`)** 함수를 정의합니다. 괄호 안이 비어 있으므로 **매개변수도 없습니다**.
- `main()`에서 `PrintHello();`를 호출하면, `PrintHello` 함수 안의 코드가 실행됩니다.
- **두 번** 호출하면 `"Hello!\n"`이 총 **6번** 출력됩니다.
- 함수는 **`main()`보다 위에** 정의해야 합니다 (또는 함수 선언을 먼저 해야 합니다).

**예상 출력**
```
Hello!
Hello!
Hello!
Hello!
Hello!
Hello!
```

---

### 4.3 매개변수가 있는 함수 – PrintHello(int size)

```c
#include <stdio.h>
// server
void PrintHello(int size) //매개변수(parameter)
{
	for(int i = 0 ; i< size ; ++i)
		printf("Hello! : %d\n", i+1);
}
// client
int main()
{
	PrintHello( 10 ); //함수의 인수(arguments)
}
```

**설명**
- `void PrintHello(int size)` : 함수가 **매개변수(parameter)** `size`를 받습니다.
- `main()`에서 `PrintHello(10);`을 호출하면, `size`에 **10**이 전달됩니다. 이때 10을 **인수(argument)**라고 합니다.
- 함수 안에서 `for` 루프가 `size`번 반복하며 `"Hello!"`를 출력합니다.
- **매개변수(parameter)**: 함수 정의에서 받는 변수 이름 (`size`)
- **인수(argument)**: 함수 호출 시 전달하는 실제 값 (`10`)

**예상 출력**
```
Hello! : 1
Hello! : 2
Hello! : 3
Hello! : 4
Hello! : 5
Hello! : 6
Hello! : 7
Hello! : 8
Hello! : 9
Hello! : 10
```

### 4.4 반환값이 있는 함수 – TotalInteger (int)

```c
#include <stdio.h>

int TotalInteger(int a, int b)
{
	int sum = 0;
	for (int i = a; i <= b; ++i)
		sum += i;
	
	return sum;
}

int main()
{
	int sum = TotalInteger(1, 10);

	printf("sum :%d\n", sum);
	printf("sum :%d\n", TotalInteger(1,10) );
	printf("sum*2 : %d\n", sum * 2);
}
```

**설명**
- `int TotalInteger(int a, int b)` : **`int`를 반환**하는 함수입니다. `a`부터 `b`까지의 합계를 구합니다.
- `return sum;` : 계산 결과를 **호출한 곳으로 돌려줍니다**.
- `int sum = TotalInteger(1, 10);` : 반환된 값을 변수 `sum`에 저장합니다.
- `printf` 안에서 `TotalInteger(1, 10)`을 직접 호출하여 반환값을 바로 출력할 수도 있습니다.
- `sum += i`는 `sum = sum + i`와 같은 뜻입니다.

**예상 출력**
```
sum :55
sum :55
sum*2 : 110
```

---

### 4.5 반환값이 있는 함수 – double 캐스팅

```c
#include <stdio.h>
double TotalInteger(int a, int b)
{
	int sum = 0;
	for (int i = a; i <= b; ++i)
		sum += i;

	return (double)sum;
}
int main()
{
	int sum = TotalInteger(1, 10);

	printf("sum : %d\n", sum);
}
```

**설명**
- 반환 타입이 `double`로 바뀌었습니다.
- `(double)sum` : 정수 `sum`을 **실수형(double)**으로 **캐스팅(형변환)**하여 반환합니다.
- `main()`에서 `int sum`으로 받으면, `double` → `int` 변환이 일어나 소수 부분은 버려집니다 (이 경우 55.0 → 55).
- 캐스팅은 `(변환할_타입)값` 형태로 씁니다.

**예상 출력**
```
sum : 55
```

## 5. 포인터 기초

**포인터**는 C언어의 가장 중요한 개념 중 하나입니다. 포인터는 **메모리 주소를 저장하는 변수**입니다.

### 5.1 문자열 포인터 맛보기

```c
#include <stdio.h>
int main()
{
	const char* s = "Hello!";

	printf("%s\n", s);
}
```

**설명**
- `const char* s = "Hello!";` : 문자열 `"Hello!"`의 **시작 주소**를 포인터 변수 `s`에 저장합니다.
- `*`이 붙은 변수는 **포인터 변수**입니다.
- `const`는 이 포인터가 가리키는 문자열을 **변경하지 않겠다**는 의미입니다.
- `%s`로 출력하면 포인터가 가리키는 주소부터 문자열을 출력합니다.
- 이 예제를 통해 포인터가 **"주소를 가리킨다"**는 개념을 맛볼 수 있습니다.

**예상 출력**
```
Hello!
```

---

### 5.2 포인터 선언과 역참조

```c
#include <stdio.h>
int main()
{
	int n = 10;
	int* p = &n;

	printf("%d %p\n", n, &n);
	printf("%d %p\n", *p, p);

	printf("size : %d\n", sizeof(p));
}
```

**설명**
- `int n = 10;` : 정수 변수 `n`에 10을 저장합니다.
- `int* p = &n;` : **포인터 변수** `p`를 선언하고, `n`의 **주소**(`&n`)를 저장합니다.
  - `&` : **주소 연산자** – 변수의 메모리 주소를 구합니다.
  - `*` (선언 시) : 이 변수가 **포인터**임을 나타냅니다.
- `*p` : **역참조(dereference)** – 포인터가 가리키는 주소의 **값**을 읽습니다. `*p`는 `n`과 같은 값(10)입니다.
- `p` : 포인터 자체의 값, 즉 `n`의 **주소**입니다.
- `%p` : **주소를 16진수**로 출력하는 형식 지정자입니다.
- `sizeof(p)` : 포인터 변수의 크기입니다 (32비트 환경: 4바이트, 64비트 환경: 8바이트).

**예상 출력 (주소는 실행마다 다름)**
```
10 0000006FF89FF714
10 0000006FF89FF714
size : 8
```

---

### 5.3 이중 포인터

```c
#include <stdio.h>
int main()
{
	int n = 10;
	int* p = &n;
	int** pp = &p;

	printf("%d %d %d\n", n, *p, **pp);
	printf("%p %p %p\n", &n, p, *pp);
}
```

**설명**
- `int** pp = &p;` : **이중 포인터**입니다. 포인터 `p`의 **주소**를 저장합니다.
- 값에 접근하는 경로:
  - `n` → 값 10 (직접)
  - `*p` → 값 10 (`p`가 가리키는 곳의 값)
  - `**pp` → 값 10 (`pp` → `p` → `n`, 두 번 역참조)
- 주소 관계:
  - `&n` : `n`의 주소
  - `p` : `n`의 주소를 담고 있음 (`&n`과 같음)
  - `*pp` : `pp`가 가리키는 값 = `p`의 값 = `n`의 주소

**메모리 구조 (개념도)**
```
pp → [p의 주소] → p → [n의 주소] → n → [10]
```

**예상 출력 (주소는 실행마다 다름)**
```
10 10 10
0000006FF89FF714 0000006FF89FF714 0000006FF89FF714
```

## 6. 값에 의한 전달 vs 주소에 의한 전달

함수에 값을 전달하는 방법에는 **값 복사**와 **주소 전달** 두 가지가 있습니다.

### 6.1 Swap 직접 구현 – 함수 없이 두 변수 교환

```c
#include <stdio.h>
int main()
{
	int a = 10;
	int b = 20;

	printf("%d %d\n", a, b);
	int t = a;
	a = b;
	b = t;
	printf("%d %d\n", a, b);
}
```

**설명**
- 두 변수의 값을 **교환(Swap)**하려면 **임시 변수** `t`가 필요합니다.
- 교환 과정:
  1. `t = a` : `a`의 값(10)을 `t`에 저장
  2. `a = b` : `b`의 값(20)을 `a`에 저장 → `a = 20`
  3. `b = t` : `t`의 값(10)을 `b`에 저장 → `b = 10`
- 이 방법은 `main()` 안에서 직접 하므로 잘 동작합니다.

**예상 출력**
```
10 20
20 10
```

---

### 6.2 Swap 함수 – 포인터를 사용한 주소 전달

```c
#include <stdio.h>
int Swap(int* pa, int* pb)
{
	int t = *pa;
	*pa = *pb;
	*pb = t;

	return 203;
}
int main()
{
	int a = 10;
	int b = 20;

	printf("%d %d\n", a, b);
	Swap(&a, &b);
	printf("%d %d\n", a, b);
}
```

**설명**
- `Swap(&a, &b)` : `a`와 `b`의 **주소**를 함수에 전달합니다.
- `int* pa, int* pb` : 포인터 매개변수로 주소를 받습니다.
- 함수 안에서 `*pa`, `*pb`를 통해 **원본 변수의 값**을 직접 수정합니다.
  - `*pa`는 `a`를 가리키므로, `*pa = *pb`는 `a = b`와 같은 효과입니다.
- 만약 포인터 대신 **값으로 전달**(`void Swap(int a, int b)`)하면, 함수 안에서 **복사본**만 바뀌고 원본은 그대로입니다. 이것이 **값에 의한 전달(call by value)**의 한계입니다.
- **주소에 의한 전달(call by address/reference)**을 사용하면 원본을 변경할 수 있습니다.

**예상 출력**
```
10 20
20 10
```

---

### 6.3 값 전달 함수 vs 주소 전달 함수

```c
#include <stdio.h>
void PrintDataValue(char c, int n, double d)
{
	printf("%d %d %g\n", c, n, d);
}
void PrintDataAddress(const char* pc, const int* pn, const double* pd)
{
	printf("%d %d %g\n", *pc, *pn, *pd);
}
int main()
{
	char c = 10;
	int n = 10;
	double d = 10;

	PrintDataValue(c, n, d);
	PrintDataAddress(&c, &n, &d);
}
```

**설명**
- `PrintDataValue(c, n, d)` : 값을 **복사해서** 전달합니다 (값에 의한 전달).
  - 함수 안에서 매개변수를 바꿔도 원본에 영향이 없습니다.
- `PrintDataAddress(&c, &n, &d)` : **주소를 전달**합니다 (주소에 의한 전달).
  - `const`가 붙은 포인터 매개변수는 **읽기 전용**이므로, 값을 수정하지 않겠다는 의미입니다.
  - `*pc`, `*pn`, `*pd`로 역참조하여 원본 값을 읽습니다.
- 두 함수 모두 **같은 결과**를 출력하지만, 내부 동작 방식이 다릅니다.

**예상 출력**
```
10 10 10
10 10 10
```

---

### 6.4 값 전달 vs 주소 전달 비교

| 구분 | 값에 의한 전달 | 주소에 의한 전달 |
|------|---------------|-----------------|
| **전달 내용** | 값의 **복사본** | 변수의 **주소** |
| **원본 수정** | 불가 | 가능 (`*포인터`로 접근) |
| **사용 예** | 값을 읽기만 할 때 | 원본을 수정해야 할 때 (Swap 등) |
| **매개변수** | `int a` | `int* pa` |

## 7. 출력 매개변수를 가진 함수

함수의 결과를 `return` 대신 **포인터 매개변수**를 통해 반환하는 방법입니다.

### 7.1 TotalInteger – 출력 매개변수 버전

```c
#include <stdio.h>
void TotalInteger(int a, int b, int* psum)
{
	int sum = 0;
	for (int i = a; i <= b; ++i)
		sum += i;

	*psum = sum;
}
int main()
{
	int sum = 0;
	
	TotalInteger(1, 10, &sum);
	printf("sum : %d\n", sum);
}
```

**설명**
- `void TotalInteger(int a, int b, int* psum)` : 반환 타입이 **`void`**이지만, 세 번째 매개변수 `int* psum`으로 결과를 돌려줍니다.
- `*psum = sum;` : 포인터를 통해 `main()`의 `sum` 변수에 **직접 값을 씁니다**.
- `TotalInteger(1, 10, &sum);` : `sum`의 주소를 전달하면, 함수가 계산 결과(55)를 `sum`에 저장합니다.
- 이 방식을 **출력 매개변수(output parameter)**라고 합니다.
- `return`으로 반환하는 것과 결과는 같지만, **여러 값을 동시에 반환**해야 할 때 출력 매개변수가 유용합니다.

**예상 출력**
```
sum : 55
```

## 8. 배열과 포인터

### 8.1 배열 선언과 반복 출력

```c
#include <stdio.h>
int main()
{
	int arr[4] = { 10,20,30,40 };

	printf("%d %d %d %d\n", arr[0], arr[1], arr[2], arr[3]);
	for (int i = 0; i < 4; ++i)
		printf("%d ", arr[i]);
	printf("\n");
}
```

**설명**
- `int arr[4] = { 10, 20, 30, 40 };` : 크기 4인 **정수 배열**을 선언하고 초기화합니다.
- 배열의 인덱스는 **0부터 시작**합니다: `arr[0]=10`, `arr[1]=20`, `arr[2]=30`, `arr[3]=40`.
- 첫 번째 `printf`에서 각 요소를 **개별적으로** 출력합니다.
- `for` 루프를 사용하면 배열 요소를 **반복으로 출력**할 수 있습니다.
- 배열 이름 `arr`은 **배열의 시작 주소**를 나타냅니다 (포인터와 밀접한 관계).

**예상 출력**
```
10 20 30 40
10 20 30 40
```

## 9. 포인터 연산

포인터에 정수를 더하거나 빼면, **가리키는 자료형의 크기 단위**로 주소가 이동합니다.

### 9.1 변수의 값과 주소

```c
#include <stdio.h>
int main()
{
	int n = 10;

	printf("%d %p\n", n, &n);
	printf("%d %p\n", n+1, &n+1);
}
```

**설명**
- `n`과 `&n` : 변수의 **값**과 **주소**를 함께 출력합니다.
- `n+1` : 값에 1을 더하므로 **11**입니다.
- `&n+1` : 주소에 1을 더하면, `int`의 크기(**4바이트**)만큼 주소가 증가합니다.
  - 예: `&n`이 `0x100`이면, `&n+1`은 `0x104`입니다 (`0x100 + 4`).
- **값의 +1**과 **주소의 +1**은 전혀 다른 의미입니다.

**예상 출력 (주소는 실행마다 다름)**
```
10 0000006FF89FF714
11 0000006FF89FF718
```

---

### 9.2 포인터 덧셈과 뺄셈

```c
#include <stdio.h>
int main()
{
	int n = 10;

	printf("%p\n", &n);
	printf("%p\n", &n + 1);
	printf("%p\n", &n + 2);
	printf("%p\n", &n + 3);
	printf("\n");
	printf("%p\n", &n);
	printf("%p\n", &n - 1);
	printf("%p\n", &n - 2);
	printf("%p\n", &n - 3);
}
```

**설명**
- `&n + 1`, `&n + 2`, `&n + 3` : 주소가 **4바이트씩 증가**합니다 (`int` = 4바이트).
- `&n - 1`, `&n - 2`, `&n - 3` : 주소가 **4바이트씩 감소**합니다.
- 포인터 연산의 단위는 **가리키는 자료형의 크기**에 따라 결정됩니다.
  - `int*`에 +1 → 주소 +4
  - `char*`에 +1 → 주소 +1
  - `double*`에 +1 → 주소 +8
- 이 원리 덕분에 배열에서 `arr + i`로 `arr[i]`의 주소에 접근할 수 있습니다.

**예상 출력 (주소는 실행마다 다름, 4바이트씩 차이 확인)**
```
0000006FF89FF714
0000006FF89FF718
0000006FF89FF71C
0000006FF89FF720

0000006FF89FF714
0000006FF89FF710
0000006FF89FF70C
0000006FF89FF708
```

| 주제 | 내용 |
|------|------|
| **조건문 – if** | `if (조건)`, `else if (조건)`, `else` |

| **for 루프** | `for (초기식; 조건식; 증감식) { ... }` |

| **continue** | 현재 반복 건너뛰기 |
| **함수 선언** | `반환타입 함수이름(매개변수) { ... }` |
| **void 함수** | 반환값 없음, `return` 생략 가능 |
| **return** | 값을 호출한 곳으로 반환 |
| **매개변수 vs 인수** | parameter(정의 시) vs argument(호출 시) |
| **포인터 선언** | `int* p = &n;` – 주소를 저장하는 변수 |
| **주소 연산자** | `&n` – 변수 `n`의 메모리 주소 |
| **역참조** | `*p` – 포인터가 가리키는 값에 접근 |
| **이중 포인터** | `int** pp = &p;` – 포인터의 주소를 저장 |
| **값 전달** | 함수에 값의 복사본 전달 (원본 불변) |
| **주소 전달** | 함수에 주소 전달 (원본 수정 가능) |
| **출력 매개변수** | 포인터 매개변수로 결과를 반환 |
| **포인터 연산** | `p + 1` → 자료형 크기만큼 주소 이동 |
| **배열과 포인터** | 배열 이름 = 시작 주소, `arr[i]` = `*(arr + i)` |
