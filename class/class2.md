## 1. 변수의 주소 (`&`와 `%p`)

C에서 모든 변수는 **메모리 어딘가에 저장**됩니다. 그 저장 위치(주소)를 확인하는 방법을 배웁니다.

### 1.1 변수 하나의 주소 출력

```c
#include <stdio.h>

int main()
{
  int a = 10;
  int b = 20;
  int result = a + b;

  printf("%p\n", &a);
}
```

**설명**
- `&a` : 변수 `a`의 **메모리 주소**를 의미합니다. `&`는 **주소 연산자(address-of operator)**입니다.
- `%p` : 주소(포인터)를 **16진수** 형태로 출력하는 형식 지정자입니다.
- 실행할 때마다 운영체제가 다른 주소를 배정하므로, 출력값은 매번 다를 수 있습니다.

**예상 출력 예시**
```
0061FF1C
```
> 실제 주소값은 환경마다 다릅니다.

---

### 1.2 여러 변수의 주소 비교

```c
#include <stdio.h>

int main()
{
  int a = 10;
  int b = 20;
  int result = a + b;

  printf("%p\n", &a);
  printf("%p\n", &b);
  printf("%p\n", &result);
}
```

**설명**
- 세 변수가 각각 **서로 다른 주소**에 저장되어 있음을 확인할 수 있습니다.
- `int`는 4바이트이므로, 일반적으로 주소가 **4씩 차이** 납니다.
- 변수 선언 순서대로 메모리에 이웃하게 배치되는 경향이 있지만, 이는 컴파일러에 따라 다를 수 있습니다.

**예상 출력 예시**
```
0061FF18
0061FF14
0061FF10
```

## 2. 산술 연산자 결과를 변수에 저장하기

1강에서 배운 산술 연산자를 복습하고, `result` 변수 활용 방법을 확인합니다.

### 2.1 연산 결과를 변수에 저장하고 출력

```c
#include <stdio.h>

int main()
{
  int a = 10;
  int b = 20;
  int result = a + b;

  printf("%d\n", result);
  printf("%d\n", a + b);
}
```

**설명**
- `int result = a + b;` : `a + b`를 계산한 값을 변수 `result`에 저장합니다.
- `printf("%d\n", result)` 와 `printf("%d\n", a+b)` 는 **동일한 결과**를 출력합니다.
- 결과를 변수에 담아두면, 나중에 그 값을 **여러 번** 재사용할 수 있어 편리합니다.

**예상 출력**
```
30
30
```

## 3. 비교 연산자

비교 연산자는 두 값을 비교하여 **참이면 1, 거짓이면 0**을 반환합니다.

### 3.1 비교 연산자 종류

```c
#include <stdio.h>

int main()
{
  // 비교 연산자
  // == != < > <= >= (true:1, false:0)

  int a = 5;
  int b = 3;

  printf("%d\n", a == b);
  printf("%d\n", a != b);
  printf("%d\n", a < b);
  printf("%d\n", a > b);
}
```

**설명**

| 연산자 | 의미 | `a=5, b=3` 결과 |
|--------|------|-----------------|
| `==` | 같다 | 0 (거짓) |
| `!=` | 다르다 | 1 (참) |
| `<` | 작다 | 0 (거짓) |
| `>` | 크다 | 1 (참) |
| `<=` | 작거나 같다 | 0 (거짓) |
| `>=` | 크거나 같다 | 1 (참) |

- C에서는 **참(true)은 1**, **거짓(false)은 0**으로 표현됩니다.
- 비교 연산의 결과를 변수에 저장하거나 `printf`로 출력할 수 있습니다.

**예상 출력**
```
0
1
0
1
```

## 4. 대입 연산자

**대입 연산자**는 변수에 값을 저장하는 연산자입니다. `=` 외에도 **복합 대입 연산자**가 있습니다.

### 4.1 단순 대입 연산자

```c
#include <stdio.h>

int main()
{
  // 대입 연산자
  int a = 5;
  int b = 3;

  a = b;
  printf("%d %d\n", a, b);
}
```

**설명**
- `a = b;` : `b`의 값을 `a`에 **복사**합니다. `b`는 그대로 3이고, `a`도 3이 됩니다.
- 수학의 `=`과 달리, C의 `=`은 **오른쪽 값을 왼쪽에 대입**하는 명령입니다.

**예상 출력**
```
3 3
```

---

### 4.2 복합 대입 연산자

```c
#include <stdio.h>

int main()
{
  // 대입 연산자
  // = += -= <<= %= ...
  int a = 5;
  int b = 3;

  a += b;
  printf("%d %d\n", a, b);
  a -= b;
  printf("%d %d\n", a, b);
  a <<= b;
  printf("%d %d\n", a, b);
  a %= b;
  printf("%d %d\n", a, b);
}
```

**설명**

| 연산자 | 의미 | 예시 (a=5, b=3) |
|--------|------|-----------------|
| `a += b` | `a = a + b` | a → 8 |
| `a -= b` | `a = a - b` | a → 5 (8-3) |
| `a <<= b` | `a = a << b` | a → 40 (5를 3비트 왼쪽 쉬프트) |
| `a %= b` | `a = a % b` | a → 1 (40 % 3) |

- 복합 대입 연산자는 **연산과 대입을 한 번에** 수행합니다.
- `a += b`는 `a = a + b`와 완전히 동일합니다.

**예상 출력**
```
8 3
5 3
40 3
1 3
```

## 5. 비트 연산자와 논리 연산자

C에는 **비트(bit) 단위**로 동작하는 비트 연산자와, **값(0 또는 비0)** 기준으로 동작하는 논리 연산자가 있습니다.

### 5.1 비트 논리 연산자 (`&`, `|`, `^`, `~`)

```c
#include <stdio.h>

int main()
{
  // 논리 연산자
  // 비트 (논리) 연산자 : AND(&) OR(|) XOR(^) NOT(~)
  int a = 5;
  int b = 3;

  printf("%d\n", a & b);
  printf("%d\n", a | b);
  printf("%d\n", a ^ b);
  printf("%d\n", ~a);
  printf("%08X\n", ~a);
  printf("%d\n", ~b);
  printf("%08X\n", ~b);
}
```

**설명**

비트 연산은 숫자를 **2진수 비트** 단위로 처리합니다.

| a=5 (2진) | b=3 (2진) | 연산 | 결과 (2진) | 결과 (10진) |
|-----------|-----------|------|------------|------------|
| 0101 | 0011 | AND (`&`) | 0001 | **1** |
| 0101 | 0011 | OR  (`\|`) | 0111 | **7** |
| 0101 | 0011 | XOR (`^`) | 0110 | **6** |
| 0101 | — | NOT (`~`) | 11…11111010 | **-6** |

- `~a` : 모든 비트를 **반전**시킵니다. 양수 `a`에 대해 `~a = -(a+1)` 입니다.
- `%08X` : 8자리 16진수로 출력합니다. `~5`는 `FFFFFFFA`로 표시됩니다.

**예상 출력**
```
1
7
6
-6
FFFFFFFA
-4
FFFFFFFC
```

---

### 5.2 값 논리 연산자 (`&&`, `||`, `!`)

```c
#include <stdio.h>

int main()
{
  // (값) 논리 연산자 : AND(&&) OR(||) NOT(!)
  int a = 0;
  int b = 3;

  printf("%d\n", a && b);
  printf("%d\n", a || b);
  printf("%d\n", !a);
  printf("%d\n", !b);
}
```

**설명**

값 논리 연산자는 **0이면 거짓, 0이 아니면 참**으로 판단합니다.

| 연산 | 의미 | a=0, b=3 결과 |
|------|------|---------------|
| `a && b` | AND : 둘 다 참이어야 참 | 0 (a가 거짓) |
| `a \|\| b` | OR : 하나라도 참이면 참 | 1 (b가 참) |
| `!a` | NOT : 참↔거짓 반전 | 1 (0이 아님 → 반전 → 1) |
| `!b` | NOT | 0 (3은 참 → 반전 → 0) |

**비트 연산자 vs 값 논리 연산자 비교**

| 구분 | 연산자 | 동작 방식 |
|------|--------|-----------|
| 비트 연산자 | `&`, `\|`, `^`, `~` | 각 비트에 대해 연산 |
| 값 논리 연산자 | `&&`, `\|\|`, `!` | 0/비0 기준으로 참/거짓 판단 |

**예상 출력**
```
0
1
1
0
```

## 6. if-else 조건문

**조건에 따라** 실행할 문장을 선택합니다.

### 6.1 기본 if-else

```c
#include <stdio.h>

int main()
{
  int a = 0;
  int b = 3;

  if (a == b)
    printf(" a == b\n");
  else
    printf(" a != b\n");
}
```

**설명**
- `if (조건)` : 조건이 **참(1)**이면 바로 아래 문장을 실행합니다.
- `else` : 조건이 **거짓(0)**이면 `else` 아래 문장을 실행합니다.
- `a = 0`, `b = 3`이므로 `a == b`는 거짓 → `" a != b"`가 출력됩니다.

**예상 출력**
```
 a != b
```

---

### 6.2 OR(`||`) 조건을 사용한 if-else

```c
#include <stdio.h>

int main()
{
  int a = 0;
  int b = 3;
  int c = 100;

  if ( (a == b) || (c == 100) )
    printf(" a == b\n");
  else
    printf(" a != b\n");
}
```

**설명**
- `(a == b) || (c == 100)` : `a == b`는 거짓(0)이지만, `c == 100`은 참(1)입니다.
- OR(`||`)는 **하나라도 참이면 참**이므로 전체가 참 → `" a == b"`가 출력됩니다.
- 조건이 복잡해질 때 괄호 `()`를 사용해 **가독성**을 높이는 것이 좋습니다.

**예상 출력**
```
 a == b
```

---

### 6.3 AND(`&&`) 조건을 사용한 if-else

```c
#include <stdio.h>

int main()
{
  int a = 0;
  int b = 3;
  int c = 100;

  if ((a == b) && (c == 100))
    printf(" a == b\n");
  else
    printf(" a != b\n");
}
```

**설명**
- `(a == b) && (c == 100)` : `a == b`는 거짓(0)이므로, AND(`&&`)는 **전체가 거짓**입니다.
- AND(`&&`)는 **둘 다 참이어야 참**이므로 → `" a != b"`가 출력됩니다.

**예상 출력**
```
 a != b
```

## 7. 쉬프트 연산자 (`<<`, `>>`)

쉬프트 연산자는 수를 **2진수 비트** 단위로 왼쪽/오른쪽으로 밀어냅니다.

### 7.1 왼쪽 쉬프트 (`<<`)

```c
#include <stdio.h>

int main()
{
  // 쉬프트(shift) 연산자
  int a = 3;

  printf("%d\n", a << 0);
  printf("%d\n", a << 1);
  printf("%d\n", a << 2);
}
```

**설명**
- `a << n` : `a`를 **왼쪽으로 n비트** 이동합니다. = `a × 2ⁿ`
  - `3 << 0` = 3 × 1 = **3**
  - `3 << 1` = 3 × 2 = **6**
  - `3 << 2` = 3 × 4 = **12**

**예상 출력**
```
3
6
12
```

---

### 7.2 오른쪽 쉬프트 (`>>`)

```c
#include <stdio.h>

int main()
{
  // 쉬프트(shift) 연산자
  int a = 10;

  printf("%d\n", a >> 0);
  printf("%d\n", a >> 1);
  printf("%d\n", a >> 2);
  printf("%d\n", a >> 3);
  printf("%d\n", a >> 4);
}
```

**설명**
- `a >> n` : `a`를 **오른쪽으로 n비트** 이동합니다. = `a ÷ 2ⁿ` (몫, 정수 나눗셈)
  - `10 >> 0` = 10 ÷ 1 = **10**
  - `10 >> 1` = 10 ÷ 2 = **5**
  - `10 >> 2` = 10 ÷ 4 = **2**
  - `10 >> 3` = 10 ÷ 8 = **1**
  - `10 >> 4` = 10 ÷ 16 = **0**

**예상 출력**
```
10
5
2
1
0
```

---

### 7.3 쉬프트 연산 정리

| 연산자 | 의미 | 수학적 의미 |
|--------|------|------------|
| `a << n` | 왼쪽으로 n비트 이동 | a × 2ⁿ |
| `a >> n` | 오른쪽으로 n비트 이동 | a ÷ 2ⁿ (몫) |

- 쉬프트 연산은 곱셈/나눗셈보다 **처리 속도가 빠릅니다**.
- 단, 오른쪽 쉬프트에서 **음수**는 환경에 따라 동작이 다를 수 있습니다.

## 8. 주소 연산 (포인터의 첫걸음) — 메인 예제

이번 ex2.cpp의 **활성화된 메인 예제**입니다.

### 8.1 변수의 값과 주소, 그리고 주소 산술

```c
#include <stdio.h>

int main()
{
  int n = 10;

  printf("%d %p\n", n, &n);
  printf("%d %p\n", n+1, &n+1);
}
```

**설명**
- `n` : 변수에 저장된 **값** (10).
- `&n` : 변수 `n`의 **메모리 주소**.
- `n+1` : 값에 1을 더한 것 → **11**.
- `&n+1` : 주소에 1을 더한 것 → 주소가 **`int` 크기(4바이트)만큼** 증가한 주소.

> `%p`로 주소를 출력할 때, `&n+1`은 `n`의 주소보다 **4바이트 앞의 주소**를 가리킵니다.  
> 이것이 나중에 배울 **포인터 산술(pointer arithmetic)**의 기초입니다.

**예상 출력 예시**
```
10 0061FF1C
11 0061FF20
```
> 주소값은 환경마다 다르지만, 두 주소의 차이는 항상 **4** (int 크기)입니다.

---

### 8.2 값 연산과 주소 연산의 차이

| 식 | 의미 | 결과 예시 |
|----|------|-----------|
| `n` | 변수의 값 | `10` |
| `n + 1` | 값에 1 더하기 | `11` |
| `&n` | 변수의 주소 | `0061FF1C` |
| `&n + 1` | 주소에 `sizeof(int)` 더하기 | `0061FF20` (+4) |

- **값에 더하는 것**과 **주소에 더하는 것**은 완전히 다릅니다.
- 주소 연산은 포인터의 핵심 개념으로, 이후 배열·포인터 단원에서 자세히 다룹니다.

| 주소 산술 | `&n + 1` = n의 주소 + sizeof(int) (4바이트 증가) |
