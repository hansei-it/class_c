## 1. 포인터와 배열의 연결

### 1.1 배열 요소의 값과 주소

```c
#include <stdio.h>
int main()
{
  int arr[4] = { 10, 20, 30, 40 };

  printf("%d\n", arr[0]);   // 10
  printf("%d\n", arr[1]);   // 20
  printf("%d\n", arr[2]);   // 30
  printf("%d\n", arr[3]);   // 40
  printf("\n");
  printf("%p\n", &arr[0]);  // 주소
  printf("%p\n", &arr[1]);  // 주소 + 4
  printf("%p\n", &arr[2]);  // 주소 + 8
  printf("%p\n", &arr[3]);  // 주소 + 12
}
```

**설명**
- `int` 배열의 각 요소는 **4바이트**씩 연속으로 저장되므로 주소가 4씩 증가합니다.

---

### 1.2 포인터로 배열 첫 번째 요소 접근

```c
#include <stdio.h>
int main()
{
  int arr[4] = { 10, 20, 30, 40 };

  int* p = &arr[0];   // p는 arr[0]의 주소를 가리킴
  printf("%d\n", *p); // 10
}
```

**설명**
- `int* p = &arr[0]` : 포인터 `p`에 배열의 첫 번째 요소 주소를 저장합니다.
- `*p` : 역참조 → `arr[0]`의 값 10을 읽습니다.

---

### 1.3 포인터를 다른 요소로 이동

```c
#include <stdio.h>
int main()
{
  int arr[4] = { 10, 20, 30, 40 };
  int* p = &arr[0];

  printf("%d\n", *p);   // 10 (arr[0])

  p = &arr[2];
  printf("%d\n", *p);   // 30 (arr[2])

  *p = 100;             // arr[2] = 100
  printf("%d %d %d %d\n", arr[0], arr[1], arr[2], arr[3]);
}
```

**예상 출력**
```
10
30
10 20 100 40
```

## 2. 포인터 산술

### 2.1 포인터에 정수 더하기

```c
#include <stdio.h>
int main()
{
  int n = 10;

  printf("%d %p", n,     &n);      // 값, 주소
  printf("%d %p", n + 1, &n + 1);  // 주소 + 4
  printf("%d %p", n + 2, &n + 2);  // 주소 + 8
  printf("%d %p", n + 3, &n + 3);  // 주소 + 12
}
```

**설명**
- `&n + 1` : `int*` 포인터에 1을 더하면 주소가 **`sizeof(int)` = 4바이트** 증가합니다.
- 포인터 산술은 **타입의 크기** 단위로 움직입니다.

---

### 2.2 포인터에서 빼기

```c
#include <stdio.h>
int main()
{
  int n = 10;

  printf("%d %p", n,     &n);      // 기준 주소
  printf("%d %p", n - 1, &n - 1);  // 주소 - 4
  printf("%d %p", n - 2, &n - 2);  // 주소 - 8
  printf("%d %p", n - 3, &n - 3);  // 주소 - 12
}
```

---

### 2.3 배열에서 포인터 산술

```c
#include <stdio.h>
int main()
{
  int arr[4] = { 10, 20, 30, 40 };
  int* p = &arr[0];

  printf("%p\n", p + 0);   // arr[0]의 주소
  printf("%p\n", p + 1);   // arr[1]의 주소 (+4)
  printf("%p\n", p + 2);   // arr[2]의 주소 (+8)
  printf("%p\n", p + 3);   // arr[3]의 주소 (+12)
}
```

**포인터 산술 정리**

| 표현 | 의미 |
|------|------|
| `p + n` | 주소 + `n × sizeof(타입)` |
| `p - n` | 주소 - `n × sizeof(타입)` |
| `int*`에서 `+1` | 주소 +4 |
| `double*`에서 `+1` | 주소 +8 |

## 3. `*(p + i)` 와 `p[i]`의 동치 관계

### 3.1 역참조 표기 (`*(p + i)`)

```c
#include <stdio.h>
int main()
{
  int arr[4] = { 10, 20, 30, 40 };
  int* p = &arr[0];

  printf("%d\n", *(p + 0));   // 10
  printf("%d\n", *(p + 1));   // 20
  printf("%d\n", *(p + 2));   // 30
  printf("%d\n", *(p + 3));   // 40
}
```

**설명**
- `*(p + i)` : `p`에서 `i`만큼 이동한 주소의 값을 읽습니다.
- 이것은 `arr[i]`와 **완전히 동일**합니다.

---

### 3.2 인덱스 표기 (`p[i]`)

```c
#include <stdio.h>
int main()
{
  int arr[4] = { 10, 20, 30, 40 };
  int* p = &arr[0];

  // *(p+i)와 p[i]는 동일
  printf("%d\n", p[0]);   // 10
  printf("%d\n", p[1]);   // 20
  printf("%d\n", p[2]);   // 30
  printf("%d\n", p[3]);   // 40
}
```

**동치 관계 정리**

| 표현 | 동등한 표현 | 의미 |
|------|------------|------|
| `arr[i]` | `*(arr + i)` | 배열 i번째 요소 |
| `p[i]` | `*(p + i)` | 포인터에서 i번째 요소 |
| `&arr[i]` | `arr + i` | i번째 요소의 주소 |

---

### 3.3 메인 예제: 포인터로 배열 출력 (두 방식 비교)

```c
#include <stdio.h>
int main()
{
  int arr[4] = { 10, 20, 30, 40 };

  int* p = &arr[0];
  printf("%d\n", *(p + 0));  // 10
  printf("%d\n", *(p + 1));  // 20
  printf("%d\n", *(p + 2));  // 30
  printf("%d\n", *(p + 3));  // 40
  printf("\n");
  printf("%d\n", p[0]);      // 10
  printf("%d\n", p[1]);      // 20
  printf("%d\n", p[2]);      // 30
  printf("%d\n", p[3]);      // 40
  printf("\n");
}
```

**예상 출력**
```
10
20
30
40

10
20
30
40

```

## 4. 포인터를 두 변수로 번갈아 가리키기

```c
#include <stdio.h>
int main()
{
  int n = 10;
  int m = 20;
  int* pn = &n;

  printf("%d %p\n", *pn, pn);   // 10, n의 주소
  pn = &m;
  printf("%d %p\n", *pn, pn);   // 20, m의 주소
}
```

**설명**
- 포인터 변수 `pn`은 처음에 `n`을 가리키다가, `pn = &m`으로 **`m`을 가리키도록 바꿀 수 있습니다**.
- 포인터가 가리키는 대상은 얼마든지 변경 가능합니다.

## 5. 포인터 크기와 타입

```c
#include <stdio.h>
int main()
{
  int n = 10;
  int* pn = &n;

  printf("%d %d\n", n, *pn);
  printf("%p %p\n", &n, pn);
  printf("%d %d\n", sizeof(n), sizeof(pn));
}
```

**설명**
- `sizeof(n)` : `int`의 크기 = **4바이트**.
- `sizeof(pn)` : 포인터의 크기 = **4바이트(32비트 환경)** 또는 **8바이트(64비트 환경)**.
- 포인터의 크기는 타입(`int*`, `double*`, `char*`)과 무관하게 **동일**합니다.

## 6. 함수 반환값과 배열 합산

### 6.1 1~10 합산 함수

```c
#include <stdio.h>
int main()
{
  int sum = 0;
  for (int i = 1; i <= 10; ++i)
    sum += i;

  printf("sum = %d\n", sum);
}
```

---

### 6.2 짝수만 합산

```c
#include <stdio.h>
int main()
{
  int sum = 0;

  for (int i = 2; i <= 10; i += 2)
    sum += i;

  printf("Sum = %d\n", sum);   // 2+4+6+8+10 = 30
}
```

---

### 6.3 조건으로 짝수만 합산

```c
#include <stdio.h>
int main()
{
  int sum = 0;

  for (int i = 1; i <= 10; i += 1)
  {
    if (i % 2 == 0)
      sum += i;
  }

  printf("Sum = %d\n", sum);   // 30
}
```

---

### 6.4 5의 배수만 합산

```c
#include <stdio.h>
int main()
{
  int sum = 0;

  for (int i = 1; i <= 10; i += 1)
  {
    if (i % 5 == 0)
      sum += i;
  }

  printf("Sum = %d\n", sum);   // 5 + 10 = 15
}
```

## 7. 배열 역순 출력 (포인터 활용)

### 7.1 끝 요소 포인터로 역순 출력

```c
#include <stdio.h>
int main()
{
  int arr[4] = { 10, 20, 30, 40 };
  int* p = &arr[3];   // 마지막 요소를 가리킴

  printf("%d\n", p[-3]);  // arr[0] = 10
  printf("%d\n", p[-2]);  // arr[1] = 20
  printf("%d\n", p[-1]);  // arr[2] = 30
  printf("%d\n", p[-0]);  // arr[3] = 40
}
```

**설명**
- `p[-n]`은 `*(p - n)`과 동일합니다. 인덱스가 **음수**일 수 있습니다.
- `p`가 `arr[3]`을 가리키고 있으므로, `p[-3]`은 `arr[3-3] = arr[0]`을 가리킵니다.

---

### 7.2 for로 역순 출력

```c
#include <stdio.h>
int main()
{
  int arr[4] = { 10, 20, 30, 40 };
  int* p = &arr[3];

  for (int i = 3; i >= 0; --i)
    printf("%d\n", p[-i]);
}
```

**예상 출력**
```
10
20
30
40
```

## 8. 정리 표

| 주제 | 내용 |
|------|------|
| 배열과 포인터 | `int* p = &arr[0];` → `p`는 배열 시작 주소 |
| 포인터 산술 | `p + n` = 주소 + `n × sizeof(타입)` |
| 역참조 | `*(p + i)` = `p[i]` = `arr[i]` (동치) |
| 배열 이름 | `arr`은 `&arr[0]`과 동일 (첫 요소 주소) |
| 음수 인덱스 | `p[-n]` = `*(p - n)` (합법적 사용 가능) |
| 포인터 크기 | 32bit: 4바이트, 64bit: 8바이트 (타입 무관) |
| 포인터로 값 변경 | `*p = 값;` → 해당 주소에 저장 |
| 짝수 합산 | `if (i % 2 == 0) sum += i;` |
| 5의 배수 합산 | `if (i % 5 == 0) sum += i;` |
