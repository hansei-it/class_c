# C언어 기초 - init.cpp : 한글 깨짐 방지 설정

이 문서는 **example/init.cpp**를 바탕으로, C언어 프로그램에서 **한글이 깨져 보이는 문제**의 원인과 해결 방법을 정리한 내용입니다.

---

## 1. 문제 상황: 한글이 깨지는 경우

Windows 환경에서 Visual Studio 등의 IDE로 C 프로그램을 실행하면, 한글을 `printf`로 출력할 때 다음과 같이 깨져 보이는 경우가 있습니다.

```
?????~!
```

또는 아무것도 출력되지 않거나, 이상한 문자가 출력됩니다.

---

## 2. 원인

| 원인 | 설명 |
|------|------|
| **소스 파일 인코딩** | 소스 파일이 UTF-8로 저장되어 있는데, 콘솔 출력은 CP949(EUC-KR) 인코딩을 사용할 때 |
| **locale 미설정** | 프로그램이 시스템의 언어/지역 설정을 사용하도록 초기화되지 않은 상태 |
| **콘솔 코드 페이지 불일치** | 프로그램 출력 인코딩과 콘솔(터미널)이 기대하는 인코딩이 다를 때 |

---

## 3. 해결 방법: `setlocale` 사용

### 3.1 init.cpp 코드

```c
#include <stdio.h>
#include <locale.h>   // <= 헤더 추가

int main()
{
  setlocale(LC_ALL, "");   // <= 현 운영체제에 맞게 locale 설정

  printf("안녕하세요~!\n");
}
```

**예상 출력**
```
안녕하세요~!
```

---

### 3.2 설명

**`#include <locale.h>`**
- `setlocale` 함수를 사용하기 위해 포함하는 헤더 파일입니다.

**`setlocale(LC_ALL, "")`**
- `setlocale` : 프로그램의 **지역화(locale)** 설정을 변경하는 함수입니다.
- `LC_ALL` : 모든 지역화 범주(숫자, 날짜, 문자 인코딩 등)를 한 번에 설정합니다.
- `""` (빈 문자열) : **현재 운영체제의 환경 설정(시스템 기본 locale)**을 그대로 사용합니다.
  - Windows의 경우 한국어 설정이라면 **CP949(EUC-KR)** 로케일로 설정됩니다.
  - 소스 파일이 CP949로 저장되어 있을 때 콘솔 출력이 일치하여 한글이 정상 표시됩니다.

**핵심 규칙**
- `setlocale(LC_ALL, "");`은 반드시 **`main` 함수의 맨 처음**, 한글을 사용하는 코드 앞에 작성합니다.

---

## 4. 인코딩 문제 원인별 해결 정리

| 상황 | 해결 방법 |
|------|-----------|
| Visual Studio에서 한글 깨짐 | `setlocale(LC_ALL, "");` 추가 |
| 소스 파일이 UTF-8 BOM으로 저장됨 | UTF-8 BOM으로 저장하거나, `_setmode` 사용 |
| VS Code + GCC 환경에서 깨짐 | 소스를 UTF-8로 저장, 터미널 인코딩도 UTF-8로 맞춤 |
| 여전히 깨질 경우 | 주석과 문자열을 모두 **영문으로만** 작성하고 한글 사용 최소화 |

---

## 5. `setlocale` 함수 요약

```c
char* setlocale(int category, const char* locale);
```

| 매개변수 | 설명 |
|----------|------|
| `LC_ALL` | 모든 범주 |
| `LC_CTYPE` | 문자 분류 및 변환 |
| `LC_NUMERIC` | 숫자 형식 (소수점 등) |
| `LC_TIME` | 날짜/시간 형식 |
| `""` | 시스템 기본 locale 사용 |
| `"C"` | 기본(영문) locale로 초기화 (기본값) |
| `"Korean"` | 한국어 locale 명시적 지정 (Windows) |

---

## 6. 템플릿으로 활용하기

새 프로그램을 작성할 때 아래 코드를 **시작 템플릿**으로 사용하면 한글 깨짐을 예방할 수 있습니다.

```c
#include <stdio.h>
#include <locale.h>

int main()
{
  setlocale(LC_ALL, "");

  // 여기서부터 코드 작성
  printf("안녕하세요~!\n");

  return 0;
}
```

---

이 설정은 **수업 예제에서 한글 주석이나 한글 출력이 필요할 때** `main` 맨 첫 줄에 추가하면 됩니다. 영문만 사용하는 경우에는 생략해도 무방합니다.
